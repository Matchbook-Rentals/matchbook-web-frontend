# Story: Accept or Reject Applications

## Description
A host can approve or decline rental applications. Approving creates a match. Both actions can be undone.

## Acceptance Criteria
- [ ] Host can approve an application from the application detail page
- [ ] Approving creates a match and shows a success page
- [ ] Host can decline an application
- [ ] Host can undo an approval
- [ ] Host can undo a decline
