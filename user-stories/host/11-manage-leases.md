# Story: Manage Leases

## Description
A host can create and manage lease templates for their listings, and prepare leases for matched renters.

## Acceptance Criteria
- [ ] Host can view leases at `/app/host/[listingId]/leases`
- [ ] Host can edit a lease for an approved applicant via the lease editor
- [ ] Lease is sent to the renter for signing after match

## Relevant Files
- `src/app/app/host/[listingId]/leases/page.tsx`
- `src/app/app/host/[listingId]/applications/[housingRequestId]/lease-editor/page.tsx`
